# erictsang 天气数据项目

## 项目需求

**任务**：查询2025年2月18日到4月20日全国390个城市的天气温度数据

**时间段划分**（共6段）：
1. 2月18日 - 2月28日（11天）
2. 3月1日 - 3月10日（10天）
3. 3月11日 - 3月20日（10天）
3月21日 - 3月31日（11天）
4. 4月1日 - 4月10日（10天）
4. 4月11日 - 4月20日（10天）

**统计指标**（每个时间段）：
- 最低温度均值
- 最高温度均值
- 温差均值（最高-最低）
- 平均温度均值（(最高+最低)/2）

**城市列表**：390个城市，来源于erictsang提供的Excel文件
